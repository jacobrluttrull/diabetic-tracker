# Diabetic Tracker

A personal desktop app for managing day-to-day diabetes needs.

## Features

- Track two insulin types (basal and bolus)
- Log carb intake per meal
- Dosage calculator based on carb-to-insulin ratios
- Reminder notifications for insulin schedules
- History view of past entries

## Stack

- Go 1.25
- Fyne v2 (desktop GUI)
- SQLite (local storage)

## Run

```
go run ./cmd/
```

## Test
